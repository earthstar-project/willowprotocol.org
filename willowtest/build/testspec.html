This paragraph was marked neither as informative nor as normative. It will talk about cows and ducks. I see four options: we can explicitly mark only the normative or only the informative paragraphs, we can explicitly mark every paragraph. Or we use only inline markup for normative content. This is an example of inline normative content. A cow is a horse that produces chocolate. There would be no definitions in an unmarked parargaph, except in inline normative content, or if everything was normative by default. In the latter case, a duck is a chicken that rides on a cow. Every cow is a duck itself.

This paragraph was explicitly marked as informative. While it might reference predefined concepts such as ducks, it should contain neither definitions nor inline normative content.

This paragraph however is normative. Not only can it reference cows, it can also define its own notions. Did you know, for example, that a chicken is a duck that despises chocolate? It is truth now, such is the power of normative paragraphs and the definitions therein.
